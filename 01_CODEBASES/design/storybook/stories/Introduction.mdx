# LocalBrain Design System

Welcome to the **LocalBrain Design System** - a modern, accessible, and performant design system built on industry-leading standards.

## üé® Core Principles

### OKLCH Color Space
All colors use the **OKLCH color space** (Oklab Lightness, Chroma, Hue) for perceptually uniform color representation:
- **Lightness (L)**: 0.0 (black) ‚Üí 1.0 (white) - perceptually uniform brightness
- **Chroma (C)**: 0.0 (gray) ‚Üí 0.37+ (vivid) - color intensity
- **Hue (H)**: 0-360¬∞ - color angle (230¬∞ = blue, 142¬∞ = green, 29¬∞ = orange, 347¬∞ = red)

**Benefits**:
- Predictable lightness across all hues
- Consistent contrast ratios
- Better than HSL, RGB, or Hex for design systems

### APCA Contrast Algorithm
We enforce **WCAG 3.0 APCA** (Accessible Perceptual Contrast Algorithm):
- **‚â•60 Lc** for body text (readable paragraphs)
- **‚â•75 Lc** for interactive elements (buttons, links, small text)
- More accurate than WCAG 2.1 contrast ratios
- Accounts for human perception of lightness

### WCAG 2.2 AA Compliance
All components meet **WCAG 2.2 AA** standards:
- ‚úÖ Color contrast (APCA ‚â•60 body, ‚â•75 interactive)
- ‚úÖ Touch targets (44√ó44px minimum)
- ‚úÖ Focus visible (clear focus indicators)
- ‚úÖ Reduced motion (respects `prefers-reduced-motion`)
- ‚úÖ Keyboard navigation (full keyboard support)
- ‚úÖ Screen reader support (semantic HTML + ARIA)

### Motion & Accessibility
Our motion system includes:
- **7 duration tokens** (instant ‚Üí very slow)
- **8 easing curves** (Material Design 3 + custom)
- **Automatic reduced motion** - All animations become instant when `prefers-reduced-motion: reduce` is enabled
- **3 strategies**: instant (default), fade-only, slower

## üöÄ Quick Start

### Installation
```bash
npm install @localbrain/design-system
```

### Usage (React)
```typescript
import { Button } from '@localbrain/design-system';
import '@localbrain/design-system/styles.css';

  return <Button variant="primary">Click me</Button>;
```

### Usage (CSS Variables)
```css
.my-component {
  background: var(--color-surface-bg);
  color: var(--color-surface-text);
  padding: var(--spacing-4);
  transition: all var(--motion-quick) var(--motion-ease-out);
}
```

### Usage (OKLCH)
```css
.custom-color {
  /* Direct OKLCH usage */
  background: oklch(0.97 0.02 250);  /* Light surface */
  color: oklch(0.22 0.04 262);        /* Dark text */
}
```
